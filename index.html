Introducción a JavaScript

JavaScript es un lenguaje de programación que permite crear interactividad en las páginas web. Funciona en todos los navegadores y se utiliza tanto en el Front-End como en el Back-End (Node.js).

  • Motores JavaScript: Los motores son el software que ejecuta JavaScript en el navegador. Los motores más conocidos son V8 (utilizado por Chrome) y SpiderMonkey (utilizado por Firefox).
  • Variables y Operadores: Las variables en JavaScript se pueden declarar utilizando var, let y const. Los operadores son símbolos que realizan operaciones en las variables, como + para sumar o === para comparar.

Introducción a TypeScript

TypeScript es un superconjunto de JavaScript que añade tipado estático al lenguaje, lo que permite detectar errores en tiempo de desarrollo. Los archivos TypeScript tienen la extensión .ts.

  • Tipos: TypeScript permite declarar variables con tipos específicos como string, number, boolean, y más.
  • Interfaces: Son estructuras que definen la forma de un objeto en TypeScript, ayudando a crear código más mantenible y claro.

Funciones en JavaScript

Las funciones son bloques de código que se ejecutan cuando son invocados. Existen varios tipos de funciones en JavaScript:

  • Sintaxis e invocación: Las funciones se definen con la palabra clave function o con funciones flecha (=>) en ES6. Se invocan simplemente con su nombre seguido de paréntesis.
  • Funciones anónimas: Son funciones sin nombre que se asignan a variables o se pasan como argumentos.
  • Funciones flecha: Introducidas en ES6, permiten escribir funciones de forma más concisa.
  • Funciones auto-invocadas: Son funciones que se ejecutan inmediatamente después de ser declaradas.

Arrays y Objetos en JavaScript

Arrays son estructuras de datos que permiten almacenar múltiples valores en una sola variable. JavaScript también soporta arrays multidimensionales, lo que permite tener arrays dentro de arrays.

  • Arrays unidimensionales: Son listas simples de elementos como [1, 2, 3].
  • Arrays multidimensionales: Son listas que contienen otras listas como [[1, 2], [3, 4]].
  • Objetos: En JavaScript, los objetos son colecciones de pares clave-valor que permiten modelar datos más complejos.
  • Map y Set: JavaScript incluye estas dos nuevas estructuras de datos que facilitan el manejo de colecciones de datos únicos o mapeados.
